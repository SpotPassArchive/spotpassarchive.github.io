SpotPass Archival Project / Wii U Guide (CFW)

Prerequisites:

  • Any modded Wii U system running Aroma or Tiramisu
  • An SD card formatted to FAT32 or exFAT
  1. Download the latest release of Dumpling from the Homebrew App Store.
    • You can also download it directly from GitHub if you don't have the Homebrew App Store.
  2. Launch Dumpling from the HOME Menu or Homebrew Launcher.
  3. Once Dumpling is open, use the D-Pad and A button to navigate; select the option that says "Dump all SpotPass data".
  4. Select your SD card if it isn't already selected, then select "OK".
  5. When finished dumping, Dumpling will ask if you want to upload the files to the SpotPass Archival Project. Select "Yes".
    • If it is unable to upload your cache to the internet, the task.db files will be located on your SD card in the /Dumpling/SpotPass Files/ folder.
  6. If your system was unable to upload your files, you can also submit them using the green link below.
