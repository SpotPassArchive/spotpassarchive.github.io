SpotPass Archival Project / 3DS ガイド (CFW)

前提:

  • Luma3DS CFWがある 3DS/2DS
    • CFWの有無を確認するには、システムの電源がオフの状態で、SELECTを押しながらPOWERボタンを押します。すぐにトップスクリーンにテキストが表示されれば、CFWがあります。そうでない場合は、CFWがないことを意味し、戻ってCFW以外のオプションを選択する必要があります。
  • SDカード
  1. SpotPassDumper11の最新版をユニバーサルアップデータアプリからダウンロードしてください。HOMEメニューから実行したい場合は.cia ファイルを、Homebrewランチャーから実行したい場合は .3dsx ファイルを選択してください。
    • Universal Updaterをお持ちでない場合は、 GitHub から直接ダウンロードすることもできます。
  2. HOMEメニューかHomebrew LauncherからSpotPassDumper11を開いてください。
  3. ツールが動作するまで待ちます。下画面に情報が表示されます。
  4. お使いのシステムがインターネットに接続されている場合、SpotPassDumper11はあなたのいつの間に通信キャッシュを直接アップロードしようとします。 キャッシュをインターネットにアップロードできなかった場合、partitionA.bin ファイルはSD/microSDカードの /spotpass_cache/ フォルダに保存されます。
  5. あなたのシステムがダンプをアップロードできなかった場合、以下の緑色のリンクから送信することもできます。
